★ 프레임워크란
  - 자바스크립트만 활용하여 컴포넌트 구조를 설계할 수 있다
  - 비동기 데이터 수신이 가능하다
  - 데이터 변화를 감지하여 뷰가 갱신된다
  - jsx문법을 사용함

★ SPA (Single Page App)
   - 여러 페이지처럼 보이는데 한 페이지로 구성됨
    - 페이지 이동 시 시간이 걸리지 않는다
    - 설계가 어렵다

★ PWA (Progressive Web App)
  - html, css, js로 앱을 설계할 수 있음

★ 개발환경구축
  - NodeJS (자바스크립트 라이브러리 관리)
  - NPM (자바스크립트 설치 및 실행)
  - create-react-app (리액트 개발환경 구축)
  - barbel, webpack

★ jsx문법
  - html 유사 구조를 이용하여 스크립트 문법을 작성할 수 있다
  - barbel library가 설치되어 있어야 함
  - return 안쪽의 객체가 컴포넌트로 구성됨
  - return 안쪽의 객체는 한 개의 부모 객체로 감싸야 한다
  - return 안쪽에서는 if, for 등의 일반 스크립트는 작성할 수 없음
  - xml 문법을 따른다, 단독객체는 반드시 셀프클로징 닫기를 해야 함 <input />
  - className, defaultValue, onClick
  - 변수나 함수를 출력시에는 {}를 사용한다
  - {}안에서는 조건문이나, for 구문을 사용할 수 없다 (함수,변수만 사용 가능)

★ 리액트문법
  - class 문법
  - hooks 문법 (함수형태)
